# Final Exam Project Frontend Course

Building Covid Web Application with React.js(Vite) and Styled Components

## install and run the application

Install application and dependencies (run it just once):

```bash
npm install
```

Running application:

```bash
npm run dev
```
Search in Browser

```bash
 http://localhost:2019/
```
